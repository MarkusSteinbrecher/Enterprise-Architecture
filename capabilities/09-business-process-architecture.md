# Capability 9: Business Process Architecture

**Domain:** Business Architecture

## Description

Design, document, and optimize business processes that support business capabilities and value streams. This capability bridges strategy and execution by defining how work gets done.

## Objectives

- Document and standardize business processes across the enterprise
- Align processes with business capabilities and value streams
- Identify automation and optimization opportunities
- Enable process governance and continuous improvement

## Key Activities

- Business process identification and cataloging
- Process modelling (using BPMN or similar)
- Process hierarchy and decomposition management
- Process-to-capability mapping
- Process performance analysis
- Automation opportunity assessment
- Process standardization and harmonization
- Process governance framework definition

## Inputs

- Business capability model
- Value stream maps
- Current process documentation
- Regulatory and compliance requirements
- Performance data and benchmarks
- Stakeholder requirements

## Outputs / Deliverables

- Business process models (BPMN diagrams)
- Process catalog / repository
- Process-to-capability mapping
- Process performance reports
- Automation opportunity assessments
- Process standards and guidelines

## Maturity Indicators

| Level | Description |
|-------|-------------|
| Initial | Processes undocumented or documented inconsistently |
| Developing | Key processes documented; limited standardization |
| Defined | Process architecture established; standard notation and repository |
| Managed | Processes measured, governed, and actively improved |
| Optimizing | Intelligent process automation; continuous process optimization |

## Key Roles

- Business Architect
- Process Analyst / Engineer
- Process Owner
- Enterprise Architect
- Automation Specialist

## Tools & Techniques

- BPMN modelling tools (e.g., Signavio, Camunda, Bizagi)
- Process mining tools
- Lean and Six Sigma methodologies
- Robotic process automation (RPA) platforms
- Process simulation tools

## KPIs / Metrics

- Process documentation coverage
- Process standardization rate
- Process cycle time and efficiency
- Automation rate
- Process compliance score
- Number of process improvement initiatives

## Relationships

- **Business Capability Modelling** - Processes realize business capabilities
- **Value Stream Mapping** - Processes execute within value streams
- **Application Portfolio Management** - Applications automate processes
- **Integration Architecture** - Integration supports cross-process data flows
- **Compliance Management** - Processes must comply with regulations
