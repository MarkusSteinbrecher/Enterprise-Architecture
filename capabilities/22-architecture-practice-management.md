# Capability 22: Architecture Practice Management

**Domain:** Architecture Management

## Description

Manage the architecture team, processes, operating model, and overall practice. This capability ensures the architecture function operates effectively and delivers value to the organization.

## Objectives

- Establish and maintain an effective architecture operating model
- Define clear roles, responsibilities, and processes
- Demonstrate and communicate architecture value
- Align architecture practice with organizational needs

## Key Activities

- Architecture operating model design
- Architecture team structure and staffing
- Architecture process definition and optimization
- Architecture engagement model design (how architecture engages with projects)
- Resource allocation and capacity management
- Architecture practice performance measurement
- Budget and financial management
- Vendor and partner management for architecture services

## Inputs

- Organizational strategy and structure
- Architecture demand and workload
- Architecture maturity assessment results
- Stakeholder feedback
- Industry best practices for architecture practices
- Budget and resource constraints

## Outputs / Deliverables

- Architecture operating model
- Architecture team charter
- Architecture engagement model
- Architecture process documentation
- Resource and capacity plans
- Architecture practice performance reports
- Architecture budget and financial reports

## Maturity Indicators

| Level | Description |
|-------|-------------|
| Initial | Architecture is ad-hoc; no formal practice |
| Developing | Architecture team established; basic processes defined |
| Defined | Formal operating model; clear roles and engagement model |
| Managed | Practice performance measured; continuous improvement active |
| Optimizing | Architecture practice as a strategic differentiator; agile and adaptive |

## Key Roles

- Chief Architect
- Architecture Manager / Practice Lead
- Enterprise Architect
- HR / People Manager
- Portfolio / Program Manager

## Tools & Techniques

- Operating model frameworks
- RACI matrices
- Capacity planning tools
- Practice performance dashboards
- Engagement model templates
- Agile architecture practices

## KPIs / Metrics

- Architecture team utilization rate
- Stakeholder satisfaction with architecture
- Architecture engagement coverage (% of projects with architecture involvement)
- Architecture process cycle time
- Value delivered by architecture practice
- Team retention rate

## Relationships

- **Architecture Skills Development** - Skills built within the practice
- **Architecture Maturity Assessment** - Maturity drives practice improvements
- **Architecture Communication** - Practice communicates its value
- **Architecture Tooling** - Tools enable the practice
- **Stakeholder Management** - Practice manages stakeholder relationships
