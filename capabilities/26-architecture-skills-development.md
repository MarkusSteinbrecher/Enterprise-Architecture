# Capability 26: Architecture Skills Development

**Domain:** Architecture Management

## Description

Develop and maintain architecture skills and competencies across the team and organization. This capability ensures the architecture function has the knowledge and expertise to deliver on its mandate.

## Objectives

- Build and maintain architecture competencies across the team
- Define clear skill profiles for architecture roles
- Enable career development for architects
- Ensure the organization has the architecture talent it needs

## Key Activities

- Architecture skill framework definition
- Skill gap assessment and planning
- Training program development and delivery
- Mentoring and coaching programs
- Architecture certification support (TOGAF, cloud, etc.)
- Community of practice facilitation
- Knowledge sharing sessions
- Career path development
- Recruitment and onboarding for architecture roles

## Inputs

- Architecture role definitions
- Skill gap assessments
- Architecture maturity assessment results
- Industry skill frameworks
- Technology trends and emerging skill needs
- Team feedback and career aspirations

## Outputs / Deliverables

- Architecture skill framework / competency model
- Skill gap analysis reports
- Training and development plans
- Mentoring program structure
- Community of practice activities
- Certification achievement records
- Career path documentation

## Maturity Indicators

| Level | Description |
|-------|-------------|
| Initial | No formal skill development; learning is ad-hoc |
| Developing | Basic training available; some mentoring |
| Defined | Skill framework defined; structured development programs |
| Managed | Skill gaps actively managed; community of practice thriving |
| Optimizing | Learning culture embedded; architects as teachers; continuous growth |

## Key Roles

- Architecture Manager / Practice Lead
- Chief Architect
- Senior Architects (as mentors)
- HR / Learning and Development
- Architecture Community of Practice leads

## Tools & Techniques

- Competency frameworks (SFIA, architecture-specific)
- Learning management systems
- Mentoring platforms
- Conference and event participation
- Internal knowledge sharing platforms
- Certification programs (TOGAF, AWS, Azure, etc.)

## KPIs / Metrics

- Skill coverage rate (% of required skills present)
- Training hours per architect
- Certification achievement rate
- Community of practice participation
- Skill gap closure rate
- Employee satisfaction with development opportunities

## Relationships

- **Architecture Practice Management** - Skills built within the practice
- **Architecture Maturity Assessment** - Maturity gaps inform skill needs
- **Innovation & Emerging Technology** - New tech requires new skills
- **Architecture Knowledge Management** - Knowledge base supports learning
- **Architecture Communication** - Communication skills critical for architects
