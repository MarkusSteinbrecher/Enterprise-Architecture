# Capability 23: Architecture Communication

**Domain:** Architecture Management

## Description

Communicate architecture vision, decisions, standards, and value to diverse audiences including executives, developers, project teams, and business stakeholders. Effective communication is critical for architecture adoption and impact.

## Objectives

- Make architecture understandable and relevant to all audiences
- Build support and buy-in for architecture decisions
- Ensure architecture guidance reaches those who need it
- Demonstrate the value of architecture to the organization

## Key Activities

- Architecture communication strategy development
- Audience-specific communication planning
- Architecture presentation and storytelling
- Architecture newsletter, blog, or bulletin publication
- Architecture brown bags, lunch-and-learns, and workshops
- Executive architecture briefings
- Architecture visualization and infographic creation
- Feedback collection and communication improvement

## Inputs

- Architecture deliverables and decisions
- Stakeholder analysis and audience profiles
- Architecture strategy and roadmap
- Organizational communication channels
- Feedback from previous communications

## Outputs / Deliverables

- Architecture communication plan
- Architecture presentations (tailored by audience)
- Architecture newsletters / bulletins
- Architecture visualizations and infographics
- Executive briefing materials
- Architecture awareness campaign materials
- Communication effectiveness reports

## Maturity Indicators

| Level | Description |
|-------|-------------|
| Initial | Architecture communicated informally; limited awareness |
| Developing | Some architecture presentations and documents shared |
| Defined | Formal communication plan; regular cadence; multiple channels |
| Managed | Tailored communication by audience; feedback actively incorporated |
| Optimizing | Architecture brand established; high awareness and engagement |

## Key Roles

- Enterprise Architect
- Chief Architect
- Architecture Manager
- Architecture Communication Lead
- All architects (as communicators)

## Tools & Techniques

- Presentation tools (PowerPoint, Keynote, Prezi)
- Visualization tools (Miro, draw.io, Canva)
- Communication platforms (Slack, Teams, email)
- Storytelling techniques
- Architecture visualization frameworks (C4 model)
- Internal blogs and wikis

## KPIs / Metrics

- Architecture awareness levels (survey)
- Communication reach (views, attendance)
- Stakeholder satisfaction with communication
- Architecture guidance adoption rate
- Feedback response rates
- Number of communication activities per period

## Relationships

- **Stakeholder Management** - Communication tailored to stakeholder needs
- **Architecture Knowledge Management** - Knowledge base supports communication
- **Architecture Practice Management** - Communication demonstrates practice value
- **Architecture Modelling** - Models are key communication tools
- **Roadmap Development** - Roadmap communicated to stakeholders
