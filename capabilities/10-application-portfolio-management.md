# Capability 10: Application Portfolio Management

**Domain:** Application Architecture

## Description

Manage the lifecycle, rationalization, and optimization of the application portfolio. This capability provides visibility into the application landscape and enables informed decisions about investment, retirement, and transformation.

## Objectives

- Maintain a complete and accurate inventory of applications
- Assess application health, value, and strategic fit
- Rationalize the portfolio to reduce redundancy and complexity
- Guide application investment and retirement decisions

## Key Activities

- Application inventory management
- Application assessment (technical fitness, business value, risk)
- Application lifecycle management (plan, build, run, retire)
- Portfolio rationalization and consolidation planning
- Application-to-capability mapping
- Application-to-technology mapping
- Total cost of ownership (TCO) analysis
- Application roadmap planning

## Inputs

- Application inventory data
- Business capability model
- Technology standards and lifecycle information
- Business strategy and priorities
- Cost and performance data
- User feedback and satisfaction data

## Outputs / Deliverables

- Application portfolio catalog
- Application assessment scorecards
- Rationalization recommendations
- Application lifecycle roadmaps
- TCO reports
- Application-to-capability heat maps
- TIME/PACE classification (Tolerate, Invest, Migrate, Eliminate)

## Maturity Indicators

| Level | Description |
|-------|-------------|
| Initial | No centralized application inventory; shadow IT prevalent |
| Developing | Basic application catalog maintained; limited assessment |
| Defined | Comprehensive portfolio with regular assessments; rationalization planned |
| Managed | Portfolio actively governed; decisions driven by data and strategy |
| Optimizing | Continuous portfolio optimization; automated discovery and assessment |

## Key Roles

- Application Architect
- Enterprise Architect
- Application Portfolio Manager
- IT Asset Manager
- Application Owners

## Tools & Techniques

- Application portfolio management tools (e.g., LeanIX, Ardoq, ServiceNow)
- Assessment frameworks (TIME, Gartner)
- Heat mapping techniques
- TCO analysis
- Discovery and dependency mapping tools

## KPIs / Metrics

- Application portfolio size and growth rate
- Application health scores (average, distribution)
- Redundancy ratio (duplicate capabilities)
- Portfolio rationalization savings
- Application retirement rate
- Technical debt levels across portfolio

## Relationships

- **Business Capability Modelling** - Applications mapped to business capabilities
- **Integration Architecture** - Integration complexity across portfolio
- **Infrastructure Architecture** - Applications deployed on infrastructure
- **Current State Assessment** - Application portfolio is part of baseline
- **Roadmap Development** - Portfolio decisions feed the roadmap
