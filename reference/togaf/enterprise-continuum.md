# TOGAF Enterprise Continuum

## Overview

The Enterprise Continuum is a classification mechanism for architecture and solution assets, ranging from highly generic (foundation) to organization-specific. It provides context for understanding how architectures evolve and relate to each other.

## Key Concepts

> To be populated with detailed research.

- **Architecture Continuum** — Classification of reusable architecture assets from foundation to organization-specific
- **Solutions Continuum** — Corresponding classification of solution implementations
- **Architecture Repository** — Structured storage for architecture outputs, organized by the continuum

## Architecture Repository Structure

- **Architecture Metamodel** — Defines the structure for architecture description
- **Architecture Landscape** — Baseline and target architectures at strategic, segment, and capability levels
- **Standards Information Base** — Applicable standards, including regulatory and industry
- **Reference Library** — Templates, patterns, and reference architectures
- **Governance Log** — Decision records and compliance assessments
