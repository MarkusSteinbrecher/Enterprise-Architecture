# TOGAF Architecture Capability Framework

## Overview

The Architecture Capability Framework provides guidance on establishing and operating an enterprise architecture practice within an organization. It defines the organizational structures, roles, skills, and processes needed to deliver effective architecture.

## Key Components

> To be populated with detailed research.

- **Architecture Board** — Governance body overseeing architecture compliance and direction
- **Architecture Compliance** — Processes for ensuring implementations conform to architecture
- **Architecture Contracts** — Agreements between development partners and sponsors on deliverables and quality
- **Architecture Governance** — Practice and orientation ensuring architecture is managed effectively
- **Architecture Maturity Models** — Assessment of organizational architecture capability maturity
- **Architecture Skills Framework** — Roles, skills, and experience required for EA practitioners
