# TOGAF Reference Documentation

## Purpose

This directory contains reference documentation on TOGAF (The Open Group Architecture Framework) as it relates to our analysis of Enterprise Architecture for the Agentic Organisation.

TOGAF serves as the **baseline framework** against which we assess how EA capabilities must evolve when AI agents become first-class participants in the enterprise.

## How We Use TOGAF

1. **Structural reference**: TOGAF's ADM phases and deliverables provide the structure for mapping our 30 EA capabilities
2. **Gap analysis baseline**: Traditional TOGAF practices serve as the "before" state against which we identify what changes for agentic organisations
3. **Vocabulary alignment**: Using TOGAF terminology ensures our findings are accessible to EA practitioners
